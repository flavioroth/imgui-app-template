Template CMake project for a single-window OpenGL / GLFW3 / ImGUI C++ Linux application.

The CMake project automatically fetches and builds its dependencies.
###### Prerequisites
Install the following X11 development libraries:
```
sudo apt install \
    libmesa-dev \
    mesa-common-dev \
    libglu1-mesa-dev \
    libxrandr \
    libxrandr-dev \
    libxinerama-dev \
    libxcursor-dev \
    libxi-dev
```
